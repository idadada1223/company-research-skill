# グループ企業の研究観点

日系大手はグループ構造を持つ企業が多い。グループ内での立ち位置を理解すると、
志望動機の説得力と「なぜ兄弟企業ではなくこの会社か」への回答が格段に強くなる。

## 典型的なグループ構造パターン

### パターン 1：持株会社型
親（持株会社）—子（事業会社）—孫会社、の階層。
**例**：NTT（持株）→ NTT ドコモ / NTT データ / NTT 東日本 / NTT コミュニケーションズ
- 持株会社は事業を直接持たず、グループ戦略・ガバナンス担当
- 事業会社は特定領域に特化
- **「持株に入ると事業会社への出向がある」ケースが多い**ため、志望時に要確認

### パターン 2：メインバンク / 商社グループ型
中核事業会社を中心に、関連会社が周辺事業を担う。
**例**：三菱グループ、三井グループ、住友グループ、伊藤忠グループ
- 明確な資本関係は薄いが、人材交流・取引関係で結びつく
- 「グループ企業」の意味が緩い場合もある

### パターン 3：垂直統合型
親会社と、サプライチェーン上の子会社群。
**例**：トヨタ自動車 → デンソー、アイシン、豊田織機
- 親会社の戦略変更がサプライヤーに直撃する
- **サプライヤー企業の志望動機で「親会社依存リスク」をどう捉えているか**は頻出

### パターン 4：業態別子会社型
親会社の事業を機能別に分社化。
**例**：伊藤忠商事 → 伊藤忠テクノソリューションズ（IT）、伊藤忠ロジスティクス（物流）
- IT 子会社・物流子会社・リース子会社などが典型
- **情報系院生が受ける「商社系 IT 子会社」「銀行系 IT 子会社」はこのパターン**

### パターン 5：ユーザー系 SIer 型（情報系院生要注意）
親会社（金融・商社・メーカー）のシステム部門が分社独立。
**例**：野村総研（野村証券）、SCSK（住友商事）、伊藤忠テクノソリューションズ（伊藤忠）、
NTT データ（NTT）、三菱 UFJ インフォメーションテクノロジー（三菱 UFJ）
- 親会社の仕事が売上の大きな割合を占める
- **「親会社案件依存度」「外販比率」は必ず確認**

## 研究時に押さえるべき観点

### A. 資本関係
- 親会社の出資比率（100% 子会社 / 連結子会社 / 持分法適用 / 業務資本提携）
- 上場しているか（独立上場 / 親子上場 / 非上場）
- **親子上場の場合、親会社が完全子会社化する可能性**を念頭に（東証は親子上場解消を推進）

### B. 事業上の役割
- グループ内で何を担当しているか（技術 / 営業 / バックオフィス / 海外展開）
- 親会社・兄弟企業との取引比率
- 外部顧客向けの売上比率（＝グループ依存度の逆数）

### C. 人材・キャリア
- グループ内出向・転籍の頻度
- グループ共通の研修・公募制度
- 親会社との給与格差（ユーザー系 SIer は親会社より低めのケースが多い）

### D. グループ再編リスク（重要）
- 直近の統合・分社・売却の動向
- 親会社の中期経営計画での位置付け（コア事業か / ノンコアか）
- **「売却・切り離しの可能性」は採用サイトには書かれない**ため、IR・ニュースで要チェック

## 面接頻出質問（グループ企業編）

以下は**ほぼ確実に聞かれる**ので準備必須：

1. **「グループ内の他社も受けていますか？」**
   - 正直に答える（嘘は一貫性の矛盾で崩れる）
   - 受けている場合は「優先順位の理由」を明確に
   - 受けていない場合は「なぜこの会社だけか」の理由を

2. **「〇〇（兄弟企業）との違いは何だと思いますか？」**
   - 事業領域・顧客層・カルチャーの違いで 3 点答える
   - IR 資料・採用サイトの違いを根拠に

3. **「親会社（〇〇）ではなく、なぜこの会社？」**（子会社志望時）
   - 専門性 / 裁量 / キャリア軸で答える
   - 「親会社の方が給料高いのに…」への回答を用意

4. **「グループ内の他社に出向の可能性がありますが OK ですか？」**
   - 出向制度があるなら、前向きに捉える姿勢を

5. **「親会社の業績が悪化したらこの会社はどうなると思いますか？」**
   - 業績連動性を理解しているか確認
   - 外販比率・独自顧客基盤に触れて答える

## 逆質問のネタ（グループ企業編）

- 「グループ全体の DX 戦略の中で、御社が担う役割は今後どう進化していきますか？」
- 「親会社との取引と外販のバランスは、中計でどのような方向性を描いていますか？」
- 「グループ内公募・出向制度はどの程度活用されていますか？具体的な事例を伺えますか？」
- 「〇〇（兄弟企業）と競合する領域は出てきていますか？どう棲み分けされていますか？」

## 典型グループの略図（参考）

### NTT グループ（情報系院生の主要志望先）
```
NTT（持株会社）
├── NTT ドコモ（モバイル）
├── NTT データ（SI・データ）※ 旧上場、現在完全子会社化
├── NTT 東日本 / 西日本（固定通信）
├── NTT コミュニケーションズ（法人・クラウド）
└── NTT 研究所（R&D）※ 院生の研究職志望先
```

### 三菱 UFJ フィナンシャル・グループ
```
MUFG（持株会社）
├── 三菱 UFJ 銀行（商業銀行）
├── 三菱 UFJ 信託銀行（信託）
├── 三菱 UFJ モルガン・スタンレー証券（証券）
└── 三菱 UFJ インフォメーションテクノロジー（IT 子会社）※ 情報系志望先
```

### トヨタグループ
```
トヨタ自動車
├── デンソー（電装品、大手部品メーカー）
├── アイシン（駆動系・車体系部品）
├── 豊田自動織機（物流機器・車両）
├── トヨタ紡織（内装）
└── Woven by Toyota（ソフトウェア子会社）※ 情報系志望先
```

### 伊藤忠グループ（総合商社）
```
伊藤忠商事
├── 伊藤忠テクノソリューションズ（CTC、IT）※ 情報系志望先
├── 伊藤忠ロジスティクス
├── ファミリーマート（連結子会社）
└── ほか約 100 社
```

これらの構造を把握した上で、**志望する会社がグループ内のどこに位置するか**を明示できると差別化になる。
