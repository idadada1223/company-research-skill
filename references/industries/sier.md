# SIer / IT サービス業界

## カテゴリ区分（必須）
SIer は「系列」で文化が大きく異なる。志望企業がどこか明記する。

| カテゴリ | 特徴 | 代表企業 |
|---|---|---|
| メーカー系 | 親会社の製品・ハードと一体で SI を提供 | 富士通、NEC、日立、東芝系 |
| ユーザー系 | 親会社（金融・商社等）のシステム子会社が独立 | NTT データ、野村総研、伊藤忠テクノ、SCSK |
| 独立系 | 特定親会社を持たない | CTC、TIS、BIPROGY |
| 外資系 | グローバルの日本法人 | アクセンチュア、IBM |
| コンサル系 | 上流工程特化 | NRI、アビーム |

## 追加で見る観点
- **上流 / 下流のどこで勝負しているか**（要件定義・設計 / 実装・保守）
- **一次請け比率**（元請けか、多重下請けの何次か）
- **主要顧客の業種構成**（金融 / 公共 / 製造業 等の偏り）
- **オフショア / ニアショア活用状況**
- **DX・クラウド・AI 投資**（AWS/Azure/GCP 認定エンジニア数等）
- **技術ブログの活発さ**（Qiita、NRI 技術ブログ等）
