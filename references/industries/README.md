## 共通の注意点（業界横断）

### メンバーシップ型 vs ジョブ型
- 日系大手は伝統的にメンバーシップ型（職能別採用、ジョブローテ前提）
- 近年ジョブ型導入の動きあり。各社で解釈・運用が異なるため要確認
- 院卒は専門性を活かしたいケースが多いため、採用形態の確認は重要

### 配属ガチャ
- 日系大手の最大懸念の一つ
- 確認すべき：「配属先は選べるか」「希望通りにならない確率」「ジョブローテ頻度」
- 配属先確約型・職種別採用の企業が増加傾向

### 情報系院生固有の注意
- **「理系総合職」は職種を絞らないため、配属リスクが高い**
- **コース別採用・職種別採用**の企業を優先する方が院卒の専門性を活かしやすい
- 研究職志望なら、研究職採用の有無・条件を要確認（多くは博士前提）
